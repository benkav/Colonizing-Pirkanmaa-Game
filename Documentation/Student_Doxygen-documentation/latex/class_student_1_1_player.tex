\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_student_1_1_player}{Player}} class represents a player in game. 

The class is used to store and access \mbox{\hyperlink{class_student_1_1_new_basic_worker}{New\+Basic\+Worker}} and Gameobject Expired weak pointers are automatically removed when requesting and moving objects

\begin{DoxyNote}{Note}
Objects are stored as weak pointers 
\end{DoxyNote}
