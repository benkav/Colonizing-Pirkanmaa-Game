\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_student_1_1_new_basic_worker}{New\+Basic\+Worker}} class represents a \char`\"{}normal worker\char`\"{} in the game. 

Worker has following production-\/efficiency\+: ~\newline

\begin{DoxyItemize}
\item Money -\/ 0.\+25 ~\newline

\item Food -\/ 1.\+00 ~\newline

\item Wood -\/ 0.\+75 ~\newline

\item Stone -\/ 0.\+50 ~\newline

\item Ore -\/ 0.\+50 ~\newline
 New\+Basic\+Workers consume Food and money. ~\newline

\item 1 Food -\/ Or \mbox{\hyperlink{class_student_1_1_new_basic_worker}{New\+Basic\+Worker}} refuses to work. ~\newline

\item 1 Money -\/ Or \mbox{\hyperlink{class_student_1_1_new_basic_worker}{New\+Basic\+Worker}} works at 50\% efficiency. ~\newline

\item Resourcefocus adds 25\% efficiency for the focused resource, even if the worker is refusing work. 
\end{DoxyItemize}
