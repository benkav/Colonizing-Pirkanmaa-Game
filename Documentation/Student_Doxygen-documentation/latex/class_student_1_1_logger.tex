\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_student_1_1_logger}{Logger}} class represents a logger in the game. 

\mbox{\hyperlink{class_student_1_1_logger}{Logger}} has following production-\/efficiency\+: ~\newline

\begin{DoxyItemize}
\item Money -\/ 0.\+50 ~\newline

\item Food -\/ 0.\+25 ~\newline

\item Wood -\/ 1.\+50 ~\newline

\item Stone -\/ 0.\+25 ~\newline

\item Ore -\/ 0.\+50 ~\newline
 Loggers consume Food and money. ~\newline

\item 1 Food -\/ Or Loggers refuses to work. ~\newline

\item 1 Money -\/ Or Loggers works at 50\% efficiency. ~\newline

\item Resourcefocus adds 25\% efficiency for the focused resource, even if the logger is refusing work. 
\end{DoxyItemize}
